## Features

- Media cutting and trimming with preview
- Media converting
- Media compressing
- Video quality downgrading
- Full customizability
- Multi-lingual
- Dark/Light mode
- Low RAM usage (thanks to Tuari)
- Privacy (everything happens locally)
